# Buni 🐰

Claude Code가 실행되는 동안 화면 위에 떠있는 macOS 픽셀아트 마스코트 앱입니다.  
Claude가 생각하고, 도구를 사용하고, 권한을 요청할 때 부니가 함께 반응합니다.

---

## 기능

### 상태 애니메이션
Claude Code의 상태에 따라 부니가 다르게 반응합니다.

| 상태 | 부니 동작 |
|------|-----------|
| 생각 중 | 뜨개질 모션 + "코드짜는중..." 말풍선 |
| 도구 사용 중 | 뜨개질 모션 + 현재 도구명 말풍선 (예: "파일 읽는 중") |
| 완료 | 대기 상태, 가끔 아이들 애니메이션 (점프, 귀 움직임) |
| 알림 | 깜짝 점프 + 알림 내용 말풍선 |
| 권한 요청 | 눈이 커지며 당근 들어올리기 + 권한 버블 팝업 |
| 유휴 (Claude 종료) | 화면 밖으로 숨기기 |

### 권한 요청 버블
Claude Code가 도구 실행 권한을 요청할 때 인터랙티브 버블이 나타납니다.

- **허용** — 해당 요청만 승인
- **전체 허용** — 이후 모든 권한 요청 자동 승인 (세션 종료까지)
- **거부** — 해당 요청 거부
- 명령어를 펼쳐서 전체 내용 확인 가능 (스크롤 지원)

### 사용량 게이지바
Claude 플랜의 5시간 슬라이딩 윈도우 사용량을 10칸 세그먼트 바로 표시합니다.

- 초록 → 노랑 → 빨강으로 색상 변화
- 사용률(%) 및 리셋까지 남은 시간 표시 (예: `↺ 4:32`)
- 5분마다 자동 갱신

### 레벨 시스템
이번 달 누적 토큰 사용량 기반으로 레벨이 올라갑니다.

- `~/.claude/projects` JSONL 파일을 스캔해 월별 input + output 토큰 합산
- 500K 토큰마다 1레벨 상승, 상한 없음
- 게이지바 위에 `★ Lv.N` 형태로 표시

### 캐릭터 선택
우클릭 컨텍스트 메뉴 → **캐릭터 변경**에서 선택할 수 있습니다.

- **부니** — 토끼
- **젤리** — 해파리

선택한 캐릭터는 앱 재시작 후에도 유지됩니다.

### 드래그로 위치 이동
부니 캐릭터를 드래그해 화면 어디에나 배치할 수 있습니다.

- 위치는 앱 재시작 후에도 유지
- **위치 초기화**: 컨텍스트 메뉴 또는 메뉴바 아이콘에서 기본 위치(오른쪽 상단)로 복원

### 클릭 통과
말풍선이 표시되지 않는 영역은 클릭이 그대로 하위 앱으로 전달됩니다.  
부니 캐릭터와 권한 버블 영역만 마우스 이벤트를 수신합니다.

### 전역 단축키
권한 요청 버블이 떠있을 때 단축키로 빠르게 응답할 수 있습니다.

- **단축키 설정**: 컨텍스트 메뉴 → **단축키 설정...** 또는 메뉴바 아이콘 → **단축키 설정...**
- 설정 가능한 단축키: 허용 / 거부 / 숨기기

### 메뉴바 아이콘
상태바 아이콘을 통해 부니를 제어할 수 있습니다.

- 부니 보이기 / 숨기기
- Claude 열기
- 위치 초기화
- 단축키 설정
- 부팅 시 자동 실행 토글
- 메뉴바 아이콘 숨기기 (이후 부니 우클릭으로만 제어)

### 자동 업데이트 알림
GitHub Releases에 새 버전이 출시되면 6시간마다 체크해 알립니다.

- 부니 말풍선 + 시스템 알림으로 새 버전 안내
- 메뉴바 클릭 → 릴리즈 페이지로 바로 이동

## 설치

### 요구사항
- macOS 13 Ventura 이상
- [Claude Code](https://claude.ai/code) CLI 설치

### 설치 방법

```bash
git clone https://github.com/EloyYang/buni.git
cd buni
bash install.sh
```

`install.sh`는 다음 작업을 수행합니다.

1. Swift 빌드
2. Claude Code 훅 스크립트 설치 (`~/.claude/companion-*.py`)
3. `~/.claude/settings.json`에 훅 자동 등록
4. 앱 실행

### 빌드 및 재설치

```bash
bash build_app.sh --install
```

## Claude Code 훅 연동

부니는 Claude Code의 훅 시스템을 통해 상태를 수신합니다.

| 훅 이벤트 | 동작 |
|-----------|------|
| `PreToolUse` | 도구 이름을 이벤트 파일에 기록 |
| `PostToolUse` | 도구 완료 이벤트 기록 |
| `Notification` | 알림 메시지 기록 |
| `Stop` | 응답 완료 이벤트 기록 |

이벤트 파일: `/tmp/claude-companion-events.jsonl`
